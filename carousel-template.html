How to Use

1. Include Files

<link rel="stylesheet" href="carousel.css">
<script src="carousel.js"></script>

2. Auto-initialization (Recommended)

<div class="carousel-container" data-carousel="my-carousel" data-auto-advance="true">
    <div class="carousel-wrapper">
        <div class="carousel-slides">
            <div class="carousel-slide active">
                <img src="image1.jpg" alt="Image 1">
                <div class="caption">Caption text</div>
            </div>
            <!-- More slides... -->
        </div>
        <button class="carousel-btn carousel-btn-prev"><span>‹</span></button>
        <button class="carousel-btn carousel-btn-next"><span>›</span></button>
    </div>
    <div class="carousel-indicators">
        <button class="indicator active"></button>
        <!-- More indicators... -->
    </div>
</div>

3. Manual Initialization

const carousel = new Carousel('#my-carousel', {
    autoAdvance: true,
    autoAdvanceTime: 5000,
    enableKeyboard: true,
    enableTouch: true,
    loop: true,
    startSlide: 0
});

// Control the carousel
carousel.next();
carousel.prev();
carousel.goTo(2);

4. Available CSS Classes

  • .carousel-small - Smaller height carousel
  • .carousel-large - Larger height carousel
  • .carousel-wide - Full width carousel
  • .carousel-auto-height - Auto-adjusting height
  • .carousel-no-captions - Hide captions

5. Data Attributes for Auto-initialization

  • data-carousel="unique-id" - Required for auto-init
  • data-auto-advance="true/false" - Enable/disable auto-advance
  • data-auto-advance-time="5000" - Time between slides (ms)
  • data-loop="true/false" - Enable/disable looping
  • data-start-slide="2" - Starting slide index
